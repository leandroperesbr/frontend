## Install

The fastest way to get __chattr__ up and running:

 * On Mac or Linux, make sure you have [nodejs][nodejs] and [npm][npm] installed

```
git clone git@github.com:dan-nl/realtimewebnode.git
cd realtimewebnode
npm install
npm start
```
